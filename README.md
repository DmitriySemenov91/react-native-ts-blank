## Basic scripts

Run iOS application:
```shell
$ npm run ios:run
```

Run Android application:
```shell
$ npm run android:run
```

Launch the start package listener:
```shell
$ npm run start
```

Run Jest tests:
```shell
$ npm run tests
```
